# 6.7960 Final Project Fall 2025
## *Can Non-Equivariant GNNs Learn to be Equivariant From Data Alone?*

Comparative analysis of SE(3)-equivariant and non-equivariant Graph Neural Networks for molecular property prediction on the QM7-X dataset.

This project uses uv.

If you want to work locally, first install uv:

MacOS:
    ```brew install uv```

Linux or MacOS Standalone:
    ```curl -LsSf https://astral.sh/uv/install.sh | sh```\
or\
    ```wget -qO- https://astral.sh/uv/install.sh | sh```\

Windows installation:
    ```powershell -c "irm https://astral.sh/uv/install.ps1 | more"```

For more information:
    [uv docs](https://docs.astral.sh/uv/)

install dependencies and the jupyter kernel by running 
    ```scripts/POSIX_install_jupyter_kernel.sh```
If you have windows, run:
    ```scripts/WINDOWS_install_jupyter_kernel.bat```
